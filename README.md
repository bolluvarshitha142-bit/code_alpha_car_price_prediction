Car Price Prediction using Machine Learning

📌 Project Overview

This project predicts the selling price of a used car using Machine Learning. A Linear Regression model is trained on historical car data to estimate the selling price based on various features.

🎯 Objective

To develop a machine learning model that accurately predicts car selling prices using Python and Scikit-learn.

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

📂 Dataset

- car data.csv

🤖 Machine Learning Algorithm

- Linear Regression

🚀 Project Workflow

1. Load the dataset.
2. Explore and understand the data.
3. Check for missing values.
4. Convert categorical data into numerical values.
5. Split the dataset into training and testing sets.
6. Train a Linear Regression model.
7. Evaluate the model using R² Score, MAE, and RMSE.
8. Predict the selling price of a car.
9. Visualize Actual vs Predicted Prices.

📊 Output

- Model Performance (R² Score)
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- Predicted Selling Price
- Actual vs Predicted Price Graph

▶️ How to Run

1. Install the required libraries:
   pip install pandas matplotlib scikit-learn
2. Place car data.csv in the project folder.
3. Run:
   python carpriceprediction.py
